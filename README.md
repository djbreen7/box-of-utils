# BOX OF UTILS

## Getting Started
* `npm install --save box-of-utils`

## Example

```typescript
import { doForDuration } from 'box-of-utils';

doForDuration((): void => console.log('hello world'), 1000);
```

* **Testing:** [Jest](https://jestjs.io/)
* **Docs:** [TypeDoc](https://typedoc.org/)
